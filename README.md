# Eaysin Mia's Portfolio & Blog

Welcome to the open-source repository for my personal portfolio and blog! This project reflects my journey, projects, and insights as a Full Stack Engineer building scalable web products, automation-heavy workflows, and AI-powered applications.

You can visit the live version of this site here: [Eaysin Mia](https://imeaysin.vercel.app/)

## 🚀 Overview

This repository holds the code for my personal workspace. It features:

- **Fast, Static Blog Performance:** Leveraging Astro's static site generation.
- **Beautiful & Modern UI:** Styled utilizing Tailwind CSS and carefully selected shadcn/ui components.
- **Content Heavy Features:** MDX support with LaTeX math rendering via KaTeX, code-block highlighting via Expressive Code.
- **Responsive Architecture:** Clean readability and aesthetic responsiveness on all devices.

## 📦 Running Locally

Want to run this locally to see how it works?

1. **Clone the repository:**

   ```bash
   git clone https://github.com/imeaysin/jainok-astro.git
   cd jainok-astro
   ```

2. **Install dependencies:**

   ```bash
   yarn install
   ```

3. **Start the development server:**

   ```bash
   yarn dev
   ```

4. **View locally:** Open your browser and visit `http://localhost:4321` (or the port specified in your terminal).
